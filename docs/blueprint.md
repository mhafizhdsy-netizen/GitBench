# **App Name**: GitAssist

## Core Features:

- GitHub Authentication: Securely authenticate users with their GitHub accounts using Supabase Auth and OAuth.
- Repository Management: Allow users to select an existing repository or create a new one directly from the app interface.
- File Upload: Enable users to upload files to their GitHub repositories using a drag-and-drop interface, with progress indicators and file validation.
- ZIP Extraction: Automatically extract ZIP files and upload their contents to the repository, handling directory structures.
- Secure Storage: Store upload records and user data securely using Supabase database.
- Code Assistant Tool: Use the 'AI Commit Message' to provide useful code commit suggestions and use it as a tool, incorporating changes to improve their code-related actions in real time.
- Error Handling: Clear and immediate user feedback for any errors during the upload process, with steps to help users.
- Batch Upload: Enable users to upload multiple files at once, improving upload efficiency.
- Pricing Section: Display pricing information, highlighting the 'Free Forever' plan and a placeholder for an 'Enterprise' plan.
- FAQ Section: Provide an accordion-style FAQ section to answer common user questions.
- About Section: Provide information about GitAssist, its mission, and the team behind it.
- Privacy Policy Page: Dedicated page outlining the privacy policy.
- Terms of Service Page: Dedicated page outlining the terms of service.
- Donation Section/Page: Integrate a donation button or section to support the development of GitAssist.
- Hero Section: Enticing Hero section with headline, subheading, description, animated background, floating cards, and CTAs.
- Features Section: Elegant cards explaining GitAssist features with icons, headings, and descriptions.
- How It Works Section: Step-by-step explanation with visual timeline or numbered cards.
- Demo/Preview Section: Animated mockup or screenshot of the GitAssist interface.
- Footer: Comprehensive footer with logo, navigation, legal links, support links, social media links and copyright notice.
- Landing Page: A combination of the Hero, Features, How it Works, and Demo sections to provide a comprehensive overview of GitAssist.
- Post-Upload Repository Selection: Allow users to select a repository after uploading a file or after successful ZIP extraction, providing flexibility in choosing the destination.

## Style Guidelines:

- Primary color: Vibrant purple (#9D4EDD) for a modern, elegant feel.
- Background color: Very dark purple-black (#0A0118) for a dark mode theme.
- Accent color: Light purple (#C77DFF) as a secondary accent to create contrast and highlights.
- Headline font: 'Space Grotesk' (sans-serif) with bold weight (700-800) for headings, reflecting technology and modernity.
- Body font: 'Inter' (sans-serif) with normal weight (400-500) for body text, to give a clean readable look.
- Code font: 'JetBrains Mono' (monospace) for displaying code snippets.
- Use Lucide React icons with a modern style. Icons incorporate the gradient purple color scheme for visual interest.
- Implement glassmorphism effect for navbar, cards, and buttons to create a modern and aesthetic UI. Navbar transforms into a pill shape on scroll.
- Employ Framer Motion for smooth transitions and animations, such as the navbar transformation and staggered feature card reveals.
